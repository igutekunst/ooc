Unit Testing
============

OOC includes unit testing for every object using the libcheck_ library.

All the tests are in `tests <https://github.com/igutekunst/ooc/tree/master/tests>`_.

 .. _libcheck: https://libcheck.github.io/check/


Running The Tests
-----------------
::

    mkdir cmake-build
    cd cmake-build
    make test_all
    ./test_all
